# File

Das File Menü stellt neben Funktionen zum [Laden der WinMF Datenformate auch Import- und Exportfunktionen](/de/help/file/loadsaveimportexport) für *.wav, *.dat und *.spk Dateien zur Verfügung. Unter [Preferences](/de/help/file/preferences) werden Pfadangaben definiert und weitere menüübergreifende Einstellungen vorgenommen. Mit Exit wird WinMF verlassen und alle Einstellungen in den Dateien default.time.mat, bzw. default.freq.mat gespeichert.
