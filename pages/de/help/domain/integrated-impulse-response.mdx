# Integrated impulse response (IIR)

Unter domain/int. impulse response wird in die Ansicht für die IIR umgeschaltet. Ist im zugehörigen Zeitsignal bereits eine berechnete IIR vorhanden, dann wird das processing übersprungen, ansonsten wird automatisch das [IIR processing menu](/de/help/edit/integrated-impulse-response-processing) aufgerufen. Soll das processing erneut ausgeführt werden, um z.B. mit geänderten Parametern neu zu rechnen, dann erfolgt das im Zeitbereich unter edit im Processing-menü für die IIR. Eine mit diesem Zeitsignal vorher berechnete IIR wird folglich überschrieben. Wird ein Zeitsignal im WinMF-Format abgespeichert, so wird die IIR in diesem File mit abgespeichert.

In der IIR Domäne sind bis auf das [Decay-times-Info Fenster](/de/help/info/decay-times), die Display-Menüs und die Plot-Funktion alle Menüs deaktiviert, da sie nicht benötigt werden.
