# Plot mode

Für den Frequenzbereich kann direkt unter dem Grafikfenster die Darstellung zwischen linearer und logarithmischer Darstellung der X-Achse umgeschaltet werden.

Im zweiten Auswahlmenü kann

im Frequenzbereich zwischen Pegel, Phase, Gruppenlaufzeit, RMS, Realteil (linear oder logarithmisch), Imaginärteil (linear oder logarithmisch) sowie einer abgewickelten Phasendarstellung (unwrapped phase) getroffen werden,

im Zeitbereich kann zwischen linearer (Amplitude) und logarithmischer (Level) Darstellung umgeschaltet werden.

Bei ``Act. channel`` wird der derzeit aktive Kanal angezeigt. Alle Werte bei den ``Cursor Infos`` werden immer für diesen Kanal ausgegeben. Die Umschaltung des aktiven Kanals erfolgt durch die Tasten 1..9, bzw. mit * wird zum nächsthöheren Kanal gewechselt und mit / zum Nächstniedrigeren.
