# Cursor tools

![cursor-tools](/cursor-tools.png)

Die Cursor tools sind eine Sammlung von Buttons, die folgende Funktionen zur Verfügung stellen:

| Funktion           | Beschreibung   |
| :----------------  | :------------ |
| ``View all (x+y)`` | Mit dieser Funktion werden die Darstellungsgrenzen von X- und Y-Achse so angepasst, dass alle Punkte der Messkurve im Display erscheinen. |
| ``View all (x)`` | Hier wird nur der Bereich der X-Achse so angepasst, dass der komplette Indexbereich dargestellt wird. |
| ``View all (y)`` | Mit dieser Funktion wird der Bereich der Y-Achse so angepasst, dass der komplette Wertebereich der Kurve im Display sichtbar ist. |
| ``Max cursor 1 (2)`` | Mit dieser Funktion wird Cursor 1 (2) auf den Punkt mit dem maximalen Wert auf der Y-Achse gesetzt, und zwar innerhalb des im Display dargestellten Bereichs. Alle Punkte außerhalb des im Display dargestellten Bereichs werden bei der Suche ignoriert. |
| ``Min cursor 1 (2)`` | Diese Funktion setzt den Cursor 1 (2) auf den Punkt im Displaybereich mit dem niedrigsten Y-Wert. |
| ``Zoom cursor 1 (2)`` | Der Bereich um Cursor 1 (2) herum wird vergrößert dargestellt mit Cursor 1 (2) in der Mitte des Displays. |
| ``Zoom out (x)`` | Der dargestellte Bereich um den aktiven Cursor herum wird verbreitert, wobei der Cursor nach Möglichkeit in der Mitte des Displays erscheint. |
| ``Exp. to 2^n`` | Der aktive Cursor wird derart verschoben, dass der Abstand der Cursor (Indexwert) die nächst größere Potenz von 2 erreicht. Im Feld Cursor 1-2 / Index wird der Wert 2^n-1 angezeigt. |
| ``Red. to 2^n`` | Der aktive Cursor wird derart verschoben, dass der Abstand der Cursor (Indexwert) die nächst kleinere Potenz von 2 erreicht. Im Feld Cursor 1-2 / Index wird der Wert 2^n-1 angezeigt. |
| ``Zoom Cursor`` | Im Display wird der Bereich zwischen den Cursorn dargestellt. |
| ``Zoom out (y)`` | Der dargestellte Bereich der Y-Achse wird vergrößert. |
| ``Cursor all`` | Es wird der ganze  X-Bereich dargestellt und die Cursor werden ebenfalls auf den ganzen Bereich eingestellt, also der linke Cursor auf das erste Sample und der rechte Cursor auf das letzte Sample. |
| ``Cursor lock`` | Der Abstand der Cursor wird fixiert. Die Cursor lassen sich nur gemeinsam parallel verschieben, solange diese Option gewählt ist. |
