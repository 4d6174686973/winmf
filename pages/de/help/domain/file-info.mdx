# File info

![file-infos](/file-infos.png)

``Name`` bezeichnet den Namen der gerade verwendeten Datei. Dieser Bezeichner dient zur Unterscheidung der gerade im Speicher befindlichen Objekte, die im Menü Time signals, bzw. Spektrums, gelistet werden. Werden Dateien importiert, also waves, dat-files oder spk-files mit den Import-Funktionen geladen, dann wird der Dateiname automatisch in Name übertragen. Im allgemeinen sind der Dateiname und Name aber nicht identisch und werden in den WinMF-Dateien im Header als separate Einträge gespeichert.

``Rate`` zeigt die Samplerate des files an.

``#Samples`` zeigt die Anzahl von Datenpunkten pro Kanal an.

``Unit`` beschreibt die physikalische Einheit der Y -Werte.

``All channels`` schaltet um zwischen der Darstellung aller Kanäle und der des aktiven Kanals.

``dB reference`` ist der Bezugswert für die logarithmische Darstellung in dB.

Mit ``Legend`` kann eine Legende in die Grafik eingefügt werden. Den einzelnen Kanälen können hier Namen zugeordnet werden.
