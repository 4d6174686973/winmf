# WinMF Display

Das Darstellungsfenster für Zeitsignale bzw. Spektren ist das zentrale Bedienelement zur Kurvenmanipulation und Einstellung der Darstellung. Mit Hilfe der Cursor können Werte abgelesen, der Darstellungsbereich eingestellt werden oder Bereiche für Edit Funktionen ausgewählt werden.

Über der Kurvendarstellung befindet sich die Kommentarzeile. Der ``Kommentar`` kann nach Anklicken beliebig editiert werden. Das Verlassen des Kommentars geht über ESC oder Klicken in das Fenster.

Unter der Kurve werden eine Reihe von Info-panels und Tools zur Verfügung gestellt. Welche Tools sichtbar sind kann im Menü ``Display/Display configuration`` eingestellt werden.
