# Export to Excel/CSV

Unter ``Write options`` werden die Einstellungen der Export-Funktion festgelegt. Mit Format wählt man zwischen Excel- und CSV-Format. Bei gewähltem Excel-Format kann außerdem das Tabellenblatt (``Excel-sheet``) und die Startzelle (``Insert at cell``) eingegeben werden. 

``Append data``

Daten werden an die nächste freie Zeile gehängt.

``Offset to cell``

Steht offset auf 0, wird bei append die nächste freie Zeile genommen, ansonsten die um den Wert bei offset verschobene Zeile. 

Use ``column header`` schreibt die Parameterdefinition in die erste Zeile der exportierten Daten.

Bei aktiviertem ``Use same name`` erhält die exportierte Datei den gleichen Namen wie die ausgewertete WinMF Datei. Anderenfalls kann unter ``Info data file`` ein beliebiger Dateiname vergeben werden. 

``Cancel`` bricht den Export ab und schließt das Fenster, während ``Ok`` den Export ausführt.
