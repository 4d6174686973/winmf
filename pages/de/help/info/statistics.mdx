# Statistics 

Im Bereich ``Info`` / ``Statistics`` können die Messungen ausgewertet werden und die ermittelten Werte exportiert werden.

## Konfiguration

Im Menü ``Select`` werden die gewünschten Parameter ausgewählt. Zusätzlich können Statistiken über die einzelnen Parameter berechnet werden.

![info-oktavbandlevel-crest-select](/info-oktavbandlevel-crest-select.gif)

Die Auswertungen erscheinen sofort im ``Info`` Fenster. Dort kann unter ``dBref`` der Bezugswert für  die dB-Werte bestimmt werden. Werden Leistungen (Power) ausgewertet, so beziehen sich diese Werte auf die für den jeweiligen Eingangskanal im Info-Fenster eingegebene Lastimpedanz (Load impedance). Die Anzeige der Lastimpedanz kann ebenfalls im Menü ``Select`` aktiviert werden. Die Impedanzwerte können dann in der Tabelle direkt editiert werden. Die im Info-Fenster verwalteten Input-bezogenen Impedanzen sind unabhängig zu den Output-bezogenen Last-Impedanzen im Frontend Menü.

Unter [Region](/de/help/edit/region-control) wird der Auswertungsbereich definiert.

Wurde vorher eine [Fract. octave Filterung](/de/help/edit/frac-oct-filtering) durchgeführt, so können bandweise die ausgewählten Werte angezeigt werden. Die Zugehörigkeit zu den Ursprungskanälen ist im Channel tag hinterlegt. Die nominelle oder auch die exakte Bandmittenfrequenz können ebenfalls in der Tabelle angezeigt werden.
