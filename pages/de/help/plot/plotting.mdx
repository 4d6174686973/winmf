# Plot

Hier kann der Inhalt der Grafikanzeigen in eine Grafikdatei geschrieben werden.

``Resolution`` bestimmt die Grafikauflösung der Datei.

``Format`` erlaubt die Auswahl zwischen verschiedenen Grafikformaten.

``Fontsize`` bestimmt die Schriftgröße der Beschriftung.

``write grafic``: es wird eine Grafikdatei erzeugt

``write matlab fig.``: es wird eine Datei im Matlab Format erstellt.

``OK`` startet den Plot, ``Cancel`` bricht den Vorgang ab. Der Plot wird unter dem gerade aktuellen Filename und Pfad mit der Endung des Grafikformats gespeichert.
