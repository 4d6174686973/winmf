# Time signals/Spectrums

Im Zeit- oder Frequenzbereichsfenster werden die geladenen Files im Menü ``Time signals``, bzw. ``Spectrums``,  verwaltet. Das gerade aktivierte Objekt wird in der Grafikausgabe als Plot angezeigt. Alle Operation, wie [Editieren](/de/help/edit), [Transformieren](/de/help/transform), [Speichern](/de/help/file), usw., wirken sich nur auf das gerade aktive Objekt aus. Wird ein neues File geladen, importiert, mit dem [Generator](/de/help/edit/generate)erzeugt, etc., dann werden die Daten des gerade aktivierten Objekts überschrieben. Auch eine [Measuring](/de/help/adda) überschreibt das gerade aktive Objekt.

In der Objektliste werden die Files mit ihrem Anzeigenamen gelistet. Werden Files importiert, so wird als Anzeigename der Filename gewählt.

Es stehen folgende Operationen zur Verfügung:
- Copy to new position: legt eine Kopie des gerade aktiven Obektes im Speicher an und aktiviert dieses Objekt
- Delete current: löscht das aktive Objekt und wechselt auf das nächste Objekt in der Liste
- Delete all curves: Löscht alle im Speicher befindlichen Objekte und legt ein leeres neues Objekt an
- New: legt ein neues leeres Objekt an, hängt es an die Liste an und aktiviert es. Für die Anzahl der in der Objektliste verwalteten Files gibt es kein Limit außer der Hauptspeichergröße.
- Copy: Kopiert die Kurven des gerade aktiven Objektes in den Paste-Speicher
- Paste: Fügt die im Paste-Speicher befindliche(n) Kurven in das gerade aktive Objekt ein. Konvertierungen von Samplerate und Linienanzahl werden bei Bedarf automatisch ausgeführt
- Spectrogram of current: öffnet das Menü zur [Erzeugung von Spektrogrammen](/de/help/time-signalsspectrums/spectrogram)
- Restore frontend setup:wenn im File ein Setup für die Einstellungen des Frontends gespeichert waren, kann das Frontend setup aus dem File in WinMF wieder hergestellt werden. Um diese Funktion zu nutzen muss es unter [Preferences](/de/help/file/preferences) aktiviert sein. Wenn kein Setup vorhanden ist, dann ist diese Funktion ausgegraut
- Restore measurement setup: wenn im File ein Setup für die letzte ausgeführte Messung gespeichert wurde, kann das Messmenü aus dem File in WinMF wieder hergestellt werden. Um diese Funktion nutzen zu können muss es unter [Preferences](/de/help/file/preferences) aktiviert sein. Wenn kein Setup vorhanden ist, dann ist diese Funktion ausgegraut
