# Macro

Macros sind ein sehr hilfreiches Werkzeug zur schnellen Durchführung immer wiederkehrender Arbeitsschritte. Komplexe Vorgänge können aufgezeichnet und beliebig oft wiederholt werden, so dass zum Beispiel mehrere Dateien mit den gleichen Prozeduren bearbeitet werden können.
