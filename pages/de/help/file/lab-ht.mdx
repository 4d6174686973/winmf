# Lab H, T

hier kann die Messung der Labortemperatur konfiguriert werden.

Die erfassten Werte für Temperatur (T) und relativer Luftfeuchte (H) werden nach jeder Messung (alle Menus in ADDA)  in den Header des aktuellen Files zusammen mit dem Zeitstempel der Messung geschrieben. Die Erfassung der Labortemperatur erfolgt nach dem Abschluss der Messung. Im Repeat Mode ist das nach Abschluss der letzten Wiederholung. Wenn der Sensor keine Werte liefert wird NaN angezeigt, wenn die konfigurierte Schnittstelle nicht ansprechbar ist (z.B. falscher Com-port) wird bei jeder Messung eine Fehlermeldung angezeigt, die Messung selbst wird aber normal durchgeführt.

Es werden folgende Sensoren/Schnittstellen unterstützt:

Arduino-USB mit SHT31 oder SHT35 Sensor, s. dazu:  [QC/SHT3X](/en/help/qc/sht3x)

Diese Optionen stehen zur Verfügung:
- ``T,H, always visible``: definiert die Sichtbarkeit im Display
- ``T,H, always plotting``: definiert ob die Werte im Plot berücksichtigt werden

Dazu s. auch die Konfiguration dieses Textfeldes: [Edit Temp, Humi](/en/help/edit/edit-temp-humi).
