# Load/Save setup as/Default values

Für das Time domain Fenster, bzw. das Frequency domain Fenster, können hier Setups geladen und gespeichert werden. Ein Setup beinhaltet alle Einstellungen und Fensterpositionen aller in diesem Fenster vorhandenen Menüs und Untermenüs. Beim Laden kann für die Domain übergreifenden Menüs (ADDA/Initialize, ADDA/Frontend setup und File/Preferences) separat entschieden werden, ob die aktuellen Settings erhalten bleiben oder durch das geladene Setting überschrieben werden sollen.

``Default values`` setzt das jeweilige Domain Fenster auf die Standardwerte zurück. Es kann auch hier entschieden werden ob die Domain übergreifenden Menüs erhalten bleiben.
