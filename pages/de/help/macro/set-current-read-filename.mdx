# Set current read filename

Setzt den ``current readfile namen`` auf den unter filename angegeben string. Mit der Option ``Use current filename`` kann der im Menu [Set current filename](/de/help/macro/set-current-filename) spezifizierte Name verwendet werden.

Dieses Menü wirkt auf die folgenden Operationen:
- [Read file channels](/de/help/edit/read-file-channels) (mit der Optione read current filename)
- read current filename file
- read current filename and insert at active cursor

Die Verwendung im Makro- oder QC Betrieb ist zunächst per Set current read filename den Filenamen anzugeben und im nächsten Schritt dann Read file channels oder read current filename file aufzurufen.
