# Start macro recording

``Start macro recording`` zeichnet die folgenden Arbeitsschritte als Macro auf. Zunächst werden Name und Dateipfad der Macrodatei festgelegt. Im gleichen Pfad werden automatisch für jeden Arbeitsschritt Setup-Dateien erzeugt, die die Einstellungen beim Aufzeichnen des Macros enthalten. Die nun folgenden Arbeitsschritte werden als Textdatei aufgezeichnet, bis mit ``Stop macro recording`` die Aufzeichnung beendet wird.

[Macrodefinitionen und Editieren von Makros](/de/help/macro/macro-definitions)
