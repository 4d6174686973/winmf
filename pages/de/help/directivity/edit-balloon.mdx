# Edit balloon

Diese Funktion dient der Darstellung von Balloons und ermöglicht den Export der Messungen. Dabei wird jeweils der dreidimensionale Balloon für eine Frequenz dargestellt. Die Export Optionen hängen von den beim Import gewählten Einstellungen ab.

## Import

Hier wird das Verzeichnis ausgewählt, in dem die Balloon-Messung in Form von Einzelmessungen im spk-Format vorliegt.

## Import options

``Angle step [°]`` beschreibt den Winkelabstand zwischen den Einzelmessungen des Balloons. Übliche Winkelauflösungen sind 10°, 5°, 2° und 1°.

Ist ``Import raw data`` aktiviert, so wird der Balloon in der vollen Frequenzauflösung der spk-Dateien eingelesen. Ist ``Import raw data`` deaktiviert, kann mit ``Bandw. [1/oct]`` die Frequenzauflösung des importierten Balloons festgelegt werden. Dabei werden die Frequenzgänge innerhalb eines Bandes im Betrag gemittelt, der Phasengang wird an der Mittenfrequenz des Bandes ohne weitere Bearbeitung verwendet.

Mit ``Frequency [Hz]`` wird die dargestellte Frequenz ausgewählt.

``Renormalize`` führt eine Renormalisierung anhand der auf dem "Nordpol" des Balloons mehrfach vorhandenen  Messpunkte durch.

``Export info`` zeigt die Export Optionen für den eingelesenen Balloon an.

## Export

[Export CLF](/de/help/directivity/clf-export) ermöglicht den Export eines importierten Balloons in verschiedenen Version des CLF Formats.
