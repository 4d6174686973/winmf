# WinSAT - System Bewertungs Tool

WinSAT ist ein peridiosch startendes Windows Programm zur Berechnung des Leistungsindexes. Wenn WinSAT läuft, werden alle Resourcen blockiert und Messungen, sowie Ansteuerungen von Robotern, unter Umständen blockiert. Es wird dringend empfohlen den periodisch geplanten Task zu deaktivieren:

``Systemsteuerung\Alle Systemsteuerungselemente\Verwaltung``
