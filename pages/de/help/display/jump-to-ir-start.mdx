# Jump to IR start

Diese Funktion sucht nach dem Beginn einer Impulsantwort und setzt den aktiven Cursor dorthin.

Mit [Select region](/de/help/edit/region-control) wird der zu durchsuchende Bereich festgelegt.

Im ``Jump to IR start setup`` wird unter ``Actual channel`` der zu durchsuchende Kanal bestimmt. ``Thresh. rel max`` legt die Pegelschwelle relativ zum Impulsmaximum fest, die als Beginn des Impulses definiert werden soll. Der Suchalgorithmus bewegt sich nun von links auf das Maximum im untersuchten Bereich zu und setzt den Cursor an die Stelle, an der die definierte Schwelle zum ersten Mal Ã¼berschritten wird.
