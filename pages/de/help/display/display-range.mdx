# Display range

Mit dieser Funktion wird der im Grafikbereich dargestellte Ausschnitt ausgewählt.

Für die X-Achse wird der Bereich wie in  [Set region](/en/help/edit/region-control) beschrieben eingestellt.

Im Frequenzbereich kann ausserdem der Bereich Y-range durch Eingabe der oberen (``Display max. [dB]``) und unteren Grenze (``Display min. [dB]``) bestimmt werden.

Im Zeitbereich wird der ``Y-range`` durch Eingabe der Werte für ``Display max.`` und ``Display min.`` bestimmt.

<Callout type="info">
  Short-cut: ``v`` setzt nur den x-range auf den im Menü definierten Bereich und schließt das Menü.
</Callout>

Die Einstellungen für das Display können auch direkt im Range-Bereich unterhalb der Anzeige durchgeführt werden.
