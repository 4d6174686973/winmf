# Display configuration

Bei ``Display configuration`` kann der Darstellungsbereich angepasst werden. Mit ``Custom linear ticks`` und ``Custom dB ticks`` kann getrennt für X- und Y-Achse ein Raster eingeblendet werden, das das Ablesen von Werten in der Grafik erleichtert.

Unter ``Display tools`` können die verschiedenen Infobereiche unterhalb der Grafik ein- bzw. ausgeschaltet werden, so dass der Grafikbereich entsprechend größer wird. Weiterhin kann unter ``Legend locat.`` und ``Legend orient.`` die Position und Ausrichtung der Legende im Grafikbereich definiert werden. ``Text interpr.`` erlaubt die Verwendung von Tex bzw. Latex Code in der Legende.
