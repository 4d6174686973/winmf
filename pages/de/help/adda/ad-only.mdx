# AD only

``AD only`` ist eine Messmethode, bei der ohne Anregungssignal ein Schallereignis (z.B. Lärm, Geräusch, Musik...) aufgezeichnet wird.

Im [Measurement setup](/de/help/adda/measurement-setup) werden die Grundeinstellungen für die Messungen vorgenommen.

Im Bereich [Processing](/de/help/adda/processing) kann die Messung bearbeitet werden.

Die Messung wird mit ``OK`` gestartet, ``Cancel`` bricht die Messung ab.

Der Screenshot zeigt eine einfache AD-Messung mit einem Kanal. Nach Messung wird die maximal erreichte Aussteuerung im Level-meter rechts unten angezeigt. Bei Übersteuerung ist der Balken rot, bis -3dBFs gelb, von -3 bis -20dBFs grün, für den restlichen Bereich bis -60dBFs blau.
