# Polar data setup

Im ``Polar data setup`` werden die Einstellungen zur Balloon- oder Directivity-Messung durchgeführt.

Es sind zahlreiche ``Presets`` hinterlegt, die häufig vorkommende Messaufgaben abdecken. Alle Presets können editiert und damit speziellen Anforderungen angepasst werden.

``El. start [°]`` / Azi. start [°] bezeichnet den Winkel, an dem die erste horizontale / vertikale Messung durchgeführt wird, ``El. stop [°]`` / ``Azi. stop [°]`` den Winkel, bei dem die letzte Messung stattfindet. Der Winkelbereich dazwischen wird in Schritten von ``El. step [°]`` / ``Azi. step [°]`` durchlaufen.

Im Bereich ``Output files`` wird festgelegt, wie mit den Messergebnissen weiter verfahren wird.

Sind ``WinMF`` oder ``SPK output`` aktiviert, so wird jede einzelne Messung im aktuellen Verzeichnis als Einzeldatei im entsprechenden Format abgespeichert.

``Copy to directivity editor`` übernimmt die Messung direkt in den Bereich [Edit directivity](/de/help/directivity/edit-directivity).

``Plot directivity`` erzeugt eine Grafik der Messung.

Die Einträge für ``Manufacturer``, ``Loudspeaker type`` und ``Configuration`` im Fenster ``Measurement info`` werden direkt in das entsprechende Fenster in [Edit directivity](/de/help/directivity/edit-directivity) übernommen.
