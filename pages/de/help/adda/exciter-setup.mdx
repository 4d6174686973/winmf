# Exciter Setup

Mit dB reference kann können die Pegelanzeigen zwischen dBV und dBU umgeschaltet werden.

Mit Level [dB] wird der Ausgangspegel des Messsystems ohne Berücksichtigung von externen Endstufen eingestellt.

Voltage [Vrms] gibt die effektive Ausgangsspannung an.

Mit Level [dBFS] wird der Pegel des Messsignals relativ zur maximalen Aussteuerung eingestellt. Der höchstmögliche Wert is 0 dBFS.

Für Messungen, die die explizite Auswahl eines Exciters verlangen, kann außerdem unter Exciter ein Messignal ausgewählt werden. Bei Frequenzgangmessungen (Frequency response, LS sensitivity und Polar response) muss für den gewählten Exciter eine passende Referenz vorliegen. Der Exciter muss für Frequenzgangmessungen eine ganze FFT-Länge haben, d. h. die Anzahl der Samples muss eine Zweierpotenz sein. Sollte keine passende Referenz gefunden werden, dann kann im jeweiligen [Referenz menü](/en/help/adda/frequency-response-reference) eine neue Referenzmessung erzeugt werden.
