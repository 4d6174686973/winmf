# Polar response

``Polar response`` Messungen stellen eine Erweiterung der  [LS sensitivity](/de/help/adda/ls-sensitivity) Messungen dar. Zur Messung des räumlichen Abstrahlverhaltens wird dazu eine Reihe von Sensitivity Messungen nacheinander in vom Lautsprecher (oder auch vom Mikrofon) aus betrachtet verschiedenen Richtungen durchgeführt. Dazu kann entweder der Lautsprecher gedreht (z.B. per Drehteller oder  [rotating arm](/de/help/adda/elf-v2)) oder das Mikrofon verschoben werden ([microphone arc](/de/help/adda/manual)). Die Darstellung erfolgt meist relativ zur Messung einer zu definierenden Hauptachse.

Liegen alle Messpunkte in einer Ebene, so spricht man je nach Lage der Ebene von einer horizontalen oder vertikalen Directivity, bei dreidimensionalen Anordnungen von einem Balloon. Die Winkelauflösung der Messungen variiert üblicherweise zwischen 10° und 1°, für nicht sehr stark richtende Lautsprecher stellen 5° meist einen guten Kompromiss zwischen Messzeit und Genauigkeit dar. Der Zusammenhang zwischen Strahlerausdehnung, Winkelauflösung und Messabstand ist beispielsweise in diesem [paper](https://ifaa-akustik.de/wp-content/uploads/2023/01/tmt-2010-ag.pdf) erklärt.

Vor der eigentlichen Messung muss mit  [Ref. - Reference measurement](/de/help/adda/frequency-response-reference) eine Referenzmessung durchgeführt werden.

Die Messeinstellungen sind zunächst identisch mit den Einstellungen in [LS sensitivity](/de/help/adda/ls-sensitivity), nur der Messmodus ``Repeat`` steht hier nicht zur Verfügung.

Zusätzlich stehen unter ``Setup`` die Menüs  [Turn table setup](/de/help/adda/turn-table-setup) und [Polar data setup](/de/help/adda/polar-data-setup) zur Verfügung.
