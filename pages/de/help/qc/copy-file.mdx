# Copy file

Es wird das unter ``Path`` liegende File mit dem Filenamen, der unter ``Copy file`` eingetragen ist, in den aktuellen Pfad des Device under Test (DUT) kopiert. Optional kann die Pfadangabe auch absolut sein. Das File kann umbenannt und mit der aktuellen Seriennummer des DUT versehen werden. Optional kann ein FAILED im QC Prozess gemeldet werden, wenn das File nicht gefunden wurde.
