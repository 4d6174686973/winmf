# User input in next menu

Dieses Menü erzwingt, dass das nachfolgende Menü im Makroprozess nicht automatisch ausgeführt wird, sondern nur das Fenster geöffnet wird. Der Prozess hält solange an, bis der User Ok in diesem Menü gedrückt hat. Dadurch können z.B., User-Eingaben in bestimmten Menüs in Makroprozessen realisiert werden.
