# Signal length

Die Signallänge wird als Gesamtzahl der Samples, als FFT-Länge oder als Länge in Sekunden angegeben. Eine FFT-Länge von N entspricht dabei einem Signal mit 2^N Samples. Die FFT-Länge muss nicht ganzzahlig sein. Die Länge in Sekunden errechnet sich aus der Samplezahl dividiert durch die Samplerate.
