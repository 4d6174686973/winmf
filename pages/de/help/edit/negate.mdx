# Negate

Mit ``Negate`` wird ein Zeitsignal mit dem Faktor -1 multipliziert, so dass der Verlauf an der X-Achse gespiegelt wird. Im Frequenzbereich wird die Phase um 180° gedreht, der Pegel bleibt unverändert.

[Select region](/de/help/edit/region-control) wählt den zu bearbeitenden Bereich aus.

[Select channel](/de/help/edit/select-channel) ermöglicht die Auswahl des zu bearbeitenden Kanals.
