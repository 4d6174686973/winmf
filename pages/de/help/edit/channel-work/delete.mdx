# Delete

``Delete`` löscht Kanäle, die höheren Kanalnummern werden entsprechend nach unten verschoben. Unter ``Delete channel`` kann die Nummer eines einzelnen zu löschenden Kanals eingegeben werden, unter ``Channel subset`` werden Kanalvektoren definiert. Die Syntax  eines Kanalvektors ist unter  [Channel subset syntax](/de/help/edit/channel-subset-syntax) beschrieben.

(Das Control ``Delete channel`` wurde aus Kompatibilitätsgründen im Makrobetrieb beibehalten)
