# Copy

Mit ``Copy`` wird der Inhalt eines Kanals oder ein Teilbereich in einen anderen Kanal hinein kopiert. Im Zielkanal vorhandene Daten werden dabei überschrieben.

[Select region](/de/help/edit/region-control) legt den zu kopierenden Teilbereich fest.

Mit ``Copy channel`` werden Quelle (``Source channel``) und Ziel (``Target channel``) für den Kopiervorgang bestimmt.
