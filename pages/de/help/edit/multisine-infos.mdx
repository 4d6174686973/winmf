# Multisine infos

Das Fenster Multisine infos zeigt Informationen über ein gerade erzeugtes [Multisinussignal](/de/help/edit/multisine-generator) an.

Die erste Information ist der Crestfaktor (Verhältnis Scheitelwert zu Effektivwert) des Multisinussignals.

In der nächsten Zeile wird die Summe aller Konflikte zwischen Anregungssignal und deren Harmonischen angezeigt. Ein Konflikt besteht dann, wenn eine Harmonische einer Frequenzlinie genau auf einer anderen Frequenzlinie liegt. Vermeiden lassen sich solche Konflikte z.B. durch längere Signale.

In der Tabelle werden in der linken Spalte alle Frequenzlinien des Multisinus angezeigt, daneben wird für jede Harmonische davon durch eine 1 ein Konflikt mit einer anderen Frequenzlinie erkennbar, eine 0 bedeutet kein Konflikt.
