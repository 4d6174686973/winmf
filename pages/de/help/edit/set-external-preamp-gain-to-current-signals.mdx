# Set external preamp gain to current signals

Diese Funktion rechnet in ein Messergebnis nachträglich eine bestimmte Einstellung für Preamp Gains rein. Das ist dann sinnvoll, wenn z.B. bei einer Messung mit WinMF versehentlich keine Preamp Gains berücksichtig wurden, oder die Messung aus einem Import stammt und keine korrekten Ergebnisse enthält. Dabei werden die gewünschten Werte im Frontend eingestellt. Die Zuordnung der Preamp Gains im Frontend und in der Datei entspricht der Nummerierung der Inputs im Frontend. Sollen beispielsweise bei Kanal 1 und Kanal 2 in der gerade im Speicher liegenden Datei Preamp Gains von 10dB für Kanal 1 und 20dB für Kanal 2 berücksichtigt werden,  so müssen im Frontend in Eingang 1 10dB und in Eingang 2 20dB eingestellt werden. Diese Funktion kann für jede Datei nur einmal durchgeführt werden, danach ist sie gesperrt.
