# Edit temp., humidity

Hier kann Aussehen und Position des Textblocks, der die Werte für Zeitstempel, T und H enthält, konfiguriert werden. Die Grundbedienung ist [here](/de/help/edit/edit-extended-comment) erläutert.

Über die Checkoxen kann gesteuert werden, welche Werte gezeigt werden und über die Auswahl ``Multiline/Single Line`` das Format des Texblocks.
