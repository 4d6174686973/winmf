# Factor control

``Factor control`` ermöglicht die Eingabe eines komplexen Zahlenwertes in verschiedenen Darstellungen. Im Frequenzbereich ist die Eingabe des Faktors Level (``Level [dB]``), Betrag und Phase (``Magnitude``, ``Phase``) oder als Real- und Imaginärteil (``Real``, ``Imag``) möglich. Die verschiedenen Darstellungen werden automatisch aus den eingegebenen Werten berechnet. Im Zeitbereich kann die Eingabe als Level oder Faktor mit Vorzeichen erfolgen. Zusätzlich kann der dB-Bezugswert eingegeben werden, um zum aktuellen Datensatz passende dB Werte erzeugen zu können.

``Copy from active cursor`` kopiert den Wert vom aktiven Cursor des aktiven Kanals in die Eingabe und gleicht außerdem den dB-Bezugswert (``dBref``) an.
