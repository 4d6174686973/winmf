# Add

``Add`` addiert einen konstanten komplexen Wert zu einem oder mehreren Kanälen. Der zu bearbeitende Bereich wird wie in  [Select region](/de/help/edit/region-control) beschrieben ausgewählt.

[Complex factor](/de/help/edit/factor-control) ermöglicht die Eingabe des Summanden.

Mit [Select channel](/de/help/edit/select-channel) wählt man den zu bearbeitenden Kanal aus.
