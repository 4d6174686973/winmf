# Set sensor sensitivities to current signals

Diese Funktion rechnet in ein Messergebnis nachträglich eine bestimmte Einstellung für die Sensitivity der benutzten Sensoren rein. Das ist dann sinnvoll, wenn z.B. bei einer Messung mit WinMF versehentlich keine Sensor Sensitivities berücksichtig wurden, oder die Messung aus einem Import stammt und keine korrekten Ergebnisse enthält. Dabei werden die gewünschten Werte im Frontend eingestellt. Die Zuordnung der Sensitivities im Frontend und in der Datei entspricht der Nummerierung der Inputs im Frontend. Sollen beispielsweise bei Kanal 1 und Kanal 2 in der gerade im Speicher liegenden Datei Sensitivities von 10mV für Kanal 1 und 20mV für Kanal 2 berücksichtigt werden,  so müssen im Frontend in Eingang 1 10mV und in Eingang 2 20mV eingestellt werden. Diese Funktion kann für jede Datei nur einmal durchgeführt werden, danach ist sie gesperrt.
