# A-weighting

Diese Funktion wendet auf einen Frequenzgang das normierte (DIN EN 61672-1:2003-10) A-Filter an. Die Phase bleibt dabei unbeeinflusst. Der Filterverlauf ist im Gegensatz zum [A-filtering (IIR)](/de/help/edit/a-filtering-iir) unabhängig von der Samplefrequenz und verläuft mittig innerhalb der Toleranzgrenzen der Norm. 

Diese Funktion kann nur einmal auf einen Frequenzgang angewendet werden und ist bis zur Durchführung von [A-unweighting](/de/help/edit/a-unweighting) geblockt.
