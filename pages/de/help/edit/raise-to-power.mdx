# Raise to power

``Raise to power`` raises time signals or spectra to the power of a real number in the time domain or a complex value in the frequency domain.

The region to be modified is selected as described in [Select region](/en/help/edit/region-control).

[Select channel](/en/help/edit/select-channel) defines the channel to be considered.

The exponent is entered under [Factor](/en/help/edit/factor-control).
