# Create curve

Mit ``Create curve`` wird aus einer Wertetabelle eine Kurve erzeugt. Die Wertetabelle kann z.B. aus einer Excel-Tabelle kopiert werden und dann per Kontextmenü in die Tabelle im ``Create curve`` Fenster eingesetzt werden. Dazu eine beliebige Zelle selektieren und per Rechtsklick das Kontextmenü aufrufen. Es werden die kompletten Daten aus dem Clipboard in die Tabelle eingesetzt. Doppelte Einträge werden gelöscht.

Mit den Einstellungen unter ``length`` werden die Parameter der Zielkurve definiert. Unter ``Curve creation setup`` können verschiedene Optionen für die Erzeugung der Kurve aus den Datenpunkten eingestellt werden.
