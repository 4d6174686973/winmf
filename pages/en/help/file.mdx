# File

The File menu provides functions for [loading files in the WinMF data formats as well as import and export functions](/en/help/file/loadsaveimportexport) for *.wav, *.dat and *.spk files. In [Preferences](/en/help/file/preferences) path definitions and further general settings can be made. By Exit WinMF is closed and all settings are stored to the files default.time.mat and default.freq.mat.
