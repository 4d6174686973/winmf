# Add

``Add`` performs the (in the frequency domain the complex) addition of two channels.

[Select region](/en/help/edit/region-control) specifies the region to be considered for the operation.

``Target = Target + Source`` describes the procedure: The (complex) summation of the specified ``Target channel`` and ``Source channel`` is written into the ``Target channel``.

``Delete source`` eliminates the specified ``Source channel`` after the operation.

``Go for all`` adds all remaining channels onto the ``Target channel``.
