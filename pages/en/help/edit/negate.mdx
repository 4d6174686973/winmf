# Negate

``Negate`` multiplies a time signal by the factor -1, so the curve is mirrored at the x-axis. In the frequency domain the phase is shifted by 180Â°, the magnitude is not modified.

[Select region](/en/help/edit/region-control) defines the segment to be modified.

[Select channel](/en/help/edit/select-channel) allows to choose the channel to be considered.
