# Known Problems

## Start after Standby or Hibernation

After restarting from standby or hibernation RME HDSP may fail. This can cause uncontrolled signals or complete fail. In this case the system has to be rebootet with hardware connected.

## Tools/Caches for SSD Optimization

SSD manufacturers often provide tools to optimize SSD performance with an additional cache. Some tools block so much resources that dop outs during measurements can occure. To check the PCI bus latency one can use this tool: dpc latency
