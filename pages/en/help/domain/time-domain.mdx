# Time domain

In the time domain representation time signals like impulse responses, excitation signals, [generated](/en/help/edit/generate) signals, etc. can be displayed and processed. In the [Time signals](/en/help/time-signalsspectrums) menu the loaded files are managed and the one to be displayed is selected.
