# Copy stat to legend

Copies the statistics of selected items from [Info/Statistics](/en/help/info/statistics) channel related into the legend.

``Copy`` to defines the target domain (Frequency, Time, or both). This option copies statistics e.g. from time domain to frequency domain or both domains.

``Copy mode`` defines if the legend is replaced (replace) or statistics is appended to the existing legend (append).

``Precision`` defines the precision of statistic values of the copied string

``Show region in legend`` copies the region of statistic evaluation into the legend
